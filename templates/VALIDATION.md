## How to Use

1. Run `/apple:validate [your-app-idea]` to generate this file through guided research
2. Or copy this template manually and fill in through your own research
3. This file informs the `/apple:new-app` command - it will reference validation findings

## Key Sections

| Section | Purpose |
|---------|---------|
| Executive Summary | Quick GO/NO-GO decision |
| Market Analysis | Is the market worth entering? |
| Competitive Landscape | Can you differentiate? |
| Revenue Potential | Is it financially viable? |
| Risk Assessment | What could go wrong? |
| Final Recommendation | Clear next steps |
