## How to Use

1. Run `/apple:verify` after completing a phase with `/apple:build`
2. The command walks through automated and manual checks
3. Results are saved to this file
4. If issues found, fix and re-verify

## Status Legend

- ✅ Pass - Works as expected
- ❌ Fail - Doesn't work, needs fix
- ⚠️ Partial - Works with issues
- N/A - Not applicable to this phase
